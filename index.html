Basics of CSS
Introduction
In this blog you learn the basics of CSS . This is for beginners who dont know about CSS styling CSS stands for Style Sheet is web document that allows you to change the appearence of HTML.
It allows you to change style of HTML.
Some basic styles are size, font styles, color, background color,margin, padding and many more are there
How CSS works
It works from top to bottom order which means that if you style something which you define in lower part of HTML will overrides the style which you defines earlier.
How CSS Applied
It applied to HTML element by declaring specific styles to each element.
Basic Syntax
selector{
property : value;
}
selector
It is either HTML element or class name or Id.
HTML element are tag names (eg: h1, p, li) to define it into CSS use it without opening and closing brackets.
For example

tag would simply be p
To style class we simply use class name with ( . ) notation

.class-name
To style id we use (#) notation followed by id names
#Id-name
property
These are predefined term that all web browser understand.
It is always followed by ( : ) eg
h1{
font-size :
}
Value
Value is particular style that assign to property
h1{
font-size : 10px;
}
Here 10px is value that assign to font-size property it gives 10px size to h1 element.

Some Basic CSS properties are

  • color : This property changes the color of text.
  • font-weight :This prperty sets bold or thin to the characters in text should be displayed.
  • background-color :This property sets the background color.
  • height :This sets the height of an element.
  • width : This sets width of an element.
  • padding :This is used to generate space around an element.
  • text-align:This property specifies the horizontal alignment of text in an element.
  • display :This property specifies the display behavioe of an element
